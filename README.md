고양이와 개를 Binary Classification하는 첫 인공지능 프로젝트입니다.

# :muscle: Motivation
>* 공부한 지식들을 사용해보고 싶었습니다.
>* 다른 사람들의 코드를 보고 클론 코딩을 하기에는 아는 게 없어서 얻는 게 없다는 판단했습니다.
>* AI 분야의 코드를 많이 봐왔지만 구조도 파악하기 어렵고, 하나부터 내가 간단한 것이라도 해보자고 생각했습니다.
>* 즉, 부족한 것들이 무엇인지 알고 싶었습니다.
>* 하고 싶은 인공지능 분야를 하기에는 그 사이에 무언가 없다고 판단했습니다. -> 간단한 프로젝트 하나 없다
>* 첫 인공지능 프로젝트로 인한 성취감을 얻기 위함입니다.
